 This console-based Java program analyzes a user-selected directory to:
   - Count Java files (.java)
   - Count files representing solved issues (files containing "solved" in the name)
 The program uses Object-Oriented Programming (OOP) principles to separate
 directory analysis logic from user interaction.

 Features:
   - Prompts user to enter a folder path (type 'quit' to exit)

   - Displays results clearly in a single-line format
